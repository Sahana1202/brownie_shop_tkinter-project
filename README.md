# 🍫 Brownie Shop Ordering System

## 📌 Project Overview

The **Brownie Shop Ordering System** is a desktop application developed using **Python Tkinter**.
This application simulates an online brownie shop where users can register, log in, browse the brownie menu, add items to a cart, generate a bill, and download the bill as a PDF.

The project demonstrates the use of GUI development in Python along with basic data handling and PDF generation.

---

## 🎯 Features

* 👤 User Registration
* 🔐 Login Authentication
* 🍫 Brownie Menu Display
* 🛒 Add Items to Cart
* 🧾 Bill Generation
* 📄 Download Bill as PDF
* 🎨 Brown-themed aesthetic user interface

---

## 🧰 Technologies Used

* Python
* Tkinter (GUI development)
* PIL / Pillow (Image handling)
* ReportLab (PDF generation)

## ▶️ How to Run the Project

1. Install Python on your system
2. Install required libraries

```
pip install pillow
pip install reportlab
```

3. Run the program

```
python main.py
```
